# Pricing

## Pricing

### Flash Pass

| Price    | Duration | Notes                                 |
| -------- | -------- | ------------------------------------- |
| $60 USDC | 1 month  | Adds one month to current expiry date |

### Classic / Elite Pass

| Timing              | Price            | Notes                                    |
| ------------------- | ---------------- | ---------------------------------------- |
| Days 1-3 of month   | $240 USDC        | Fixed early-month rate                   |
| Day 4 onwards       | $300 → $125 USDC | Decreases linearly daily until month end |
| Upfront multi-month | $240 × months    | For bulk renewals                        |

{% hint style="info" %}
Classic/Elite renewals always extend to the **1st day of the following month**.
{% endhint %}
