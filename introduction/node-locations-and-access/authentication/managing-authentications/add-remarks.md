# Add Remarks

**`!rpc remark`**

This command allows users to add remarks to their assigned slots for additional information. The bot will display the list of assigned IP slots in rows and show any existing remarks. The user can then add their own remark to a specific slot.

Enter the row number you want to add remark, then enter your remark.
